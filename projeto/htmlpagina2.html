Tags Básicas do HTML

No HTML, as tags são elementos fundamentais que definem o conteúdo e a estrutura de uma página web. Cada tag possui um propósito específico e é usada para criar diferentes tipos de conteúdo, como parágrafos, cabeçalhos, links, imagens, listas, e muito mais. Aqui estão algumas das tags básicas mais importantes e comuns no HTML.

  1. h1 a h6 - Cabeçalhos

    As tags de cabeçalho (h1 a h6) são usadas para definir títulos e subtítulos em uma página web. O h1 é o cabeçalho mais importante e maior, enquanto o h6 é o menor e menos importante.

    • h1: Geralmente usado para o título principal da página.

    • h2: Usado para seções principais.

    • h3 a h6: Usados para subseções.

    Exemplo:

                                
                            <h1>Título Principal</h1>
                            <h2>Seção</h2>
                            <h3>Subseção</h3>
                            <h4>Sub-subseção</h4>
                            <h5>Sub-sub-subseção</h5>
                            <h6>Sub-sub-sub-subseção</h6>

  2. p - Parágrafos

    A tag p é usada para definir parágrafos de texto. Cada parágrafo criado com p é automaticamente separado por um espaço em branco do parágrafo anterior e seguinte.

    Exemplo:

                            
                        <p>Este é um parágrafo de texto.</p>
                        <p>Este é outro parágrafo de texto.</p>

  3. a - Links

    A tag a é usada para criar links, que podem direcionar o usuário para outras páginas ou sites. O atributo href define o destino do link.

    Exemplo:

                            
                        <a href="https://www.exemplo.com">Visite o Exemplo</a>
                        <a href="https://www.google.com">Pesquise no Google</a>

  4. img - Imagens

    A tag img é usada para incorporar imagens em uma página web. O atributo src especifica o caminho da imagem, e o atributo alt fornece um texto alternativo para descrever a imagem, caso ela não seja carregada.

    Exemplo:

                            
                        <img src="imagem.jpg" alt="Descrição da imagem">
                        <img src="logo.png" alt="Logo da Empresa">

  5. ul, ol, e li - Listas

    O HTML permite a criação de listas ordenadas e não ordenadas:

    • ul: Define uma lista não ordenada (com marcadores).
    • ol: Define uma lista ordenada (numerada).
    • li: Define um item de lista.

    Exemplo de lista ordenada e nao ordenada

                            
                        <ul>
                          <li>Item 1</li>
                          <li>Item 2</li>
                          <li>Item 3</li>
                        </ul>
            
                        <ol>
                          <li>Primeiro item</li>
                          <li>Segundo item</li>
                          <li>Terceiro item</li>
                        </ol>

  6. div - Divisões de Página

    A tag div é um contêiner genérico usado para agrupar outros elementos e criar seções de uma página. Ela é frequentemente utilizada em conjunto com CSS para estilizar ou organizar conteúdo.

    Exemplo:

                            
                        <div>
                          <h3>Seção de Conteúdo</h3>
                          <p>Este é um parágrafo dentro de uma div.</p>
                        </div>

  7. span - Elemento em Linha

    A tag span é semelhante à div, mas é um elemento em linha, o que significa que ela não cria uma quebra de linha antes e depois de seu conteúdo. É usada principalmente para estilizar partes específicas de texto ou conteúdo dentro de outros elementos.

    Exemplo:

                            
                        <p>Este é um <span style="color: red;">texto em vermelho</span> dentro de um parágrafo.</p>
                        
    
                        <p>Outro exemplo: <span style="font-weight: bold;">texto em negrito</span>.</p>
