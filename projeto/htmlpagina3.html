Estilizando com CSS

Os seletores CSS são usados para "selecionar" os elementos HTML que você deseja estilizar. Alguns dos seletores mais comuns incluem:

  1. Seletores CSS

    2. Os seletores CSS são usados para "selecionar" os elementos HTML que você deseja estilizar. Alguns dos seletores mais comuns incluem:

    • Seletores de Tipo: Aplica estilos a todos os elementos de um tipo específico, como todos os p ou h1.

      Exemplo: p { color: blue; } — Isso altera a cor de todos os parágrafos (p) para azul.

    • Seletores de Classe: Aplica estilos a elementos que possuem uma classe específica, definida no atributo class.

      Exemplo: .meu-estilo { font-size: 20px; } — Isso altera o tamanho da fonte para 20px em todos os elementos com a classe "meu-estilo".

    • Seletores de ID: Aplica estilos a um elemento único que possui um ID específico, definido no atributo id.

      Exemplo: #meu-id { background-color: yellow; } — Isso altera o fundo do elemento com o ID "meu-id" para amarelo.

  2. Propriedades e Valores

    3. No CSS, cada estilo é definido por uma propriedade e um valor. Por exemplo, a propriedade color define a cor do texto, e a propriedade background-color define a cor de fundo.

    • Cor do Texto:

      color: red; — Define a cor do texto como vermelho.

    • Cor de Fundo:

      background-color: lightgray; — Define a cor de fundo como cinza claro.

    • Tamanho da Fonte:

      font-size: 18px; — Define o tamanho da fonte como 18 pixels.

  3. Aplicando CSS Interno

    O CSS pode ser aplicado diretamente no cabeçalho de uma página HTML usando a tag style. Isso é conhecido como CSS interno. Veja um exemplo básico:

                                
                            <style>
                                body {
                                    background-color: #f0f0f0; /* Cor de fundo da página */
                                }
                                h1 {
                                    color: #333; /* Cor do texto do cabeçalho */
                                    font-size: 24px; /* Tamanho da fonte do cabeçalho */
                                }
                                p {
                                    font-family: Arial, sans-serif; /* Fonte usada para o parágrafo */
                                    color: #555; /* Cor do texto do parágrafo */
                                }
                            </style>

  4. Estilizando com Classes e IDs

    Ao aplicar classes e IDs aos seus elementos HTML, você pode estilizar de forma específica. Por exemplo:

    • Classe:

      HTML

                                      
                                <p class="destaque">Este parágrafo é destacado com CSS.</p>

      CSS

                                      
                                .destaque {
                                    color: blue;
                                    font-weight: bold;
                                }

    • ID:

      HTML

                                      
                                <h1 id="titulo-principal">Este é o título principal</h1>

      CSS

                                      
                                #titulo-principal {
                                    color: green;
                                    text-align: center;
                                }
